 Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI.
The objective is to analyze a real-world dataset, extract meaningful insights, and present them through interactive dashboards and business-ready reports.

The project covers:

* Data loading and exploration using Python
* Data cleaning and preprocessing
* SQL-based analysis using relational databases
* Dashboard development in Power BI
* Insight reporting and presentation using Gamma

* Source: CSV file (can be replaced with any similar dataset)
